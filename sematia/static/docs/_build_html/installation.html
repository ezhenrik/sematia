Note

These instructions have only been tested on Mac OS X and Linux.

Sematia is running on Flask, a Python microframework. To install Flask and Sematia, you need an up-to-date Python 2.x installation (Sematia doesn’t yet support Python 3). Sematia also requires a working MySQL (or MariaDB) installation.

First, create the virtual environment:

$ virtualenv  ~/Virtualenvs/sematia

$ source venv/bin/activate

The Sematia source is available at GitHub. Clone the repository and edit the config.py file according to your system configuration.

Then, start Sematia with the following command:

$ python run.py

Sematia will now be running on http://localhost:5000.
